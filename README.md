## Installation

SKSwiftExtension is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "SKSwiftExtension", git: "git@github.com:sehkai/SKSwiftExtension.git"
```
